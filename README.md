# Computational Biology Projects

This repository contains small Python projects that explore biological data analysis and introductory machine learning.  
Each notebook demonstrates how programming and biology can come together to extract insights from data.

## Projects
1. **Amino Acid Frequency Analysis**  
   Analyzes protein FASTA sequences to calculate amino acid composition and visualize frequency distributions.
